<p>Let's build an interesting data structure straight out of an interview programming challenge from
<a href="https://stripe.com/" rel="nofollow">Stripe</a>.
This is more of an algorithms challenge than a Python challenge, but we hope you're still

<p>At a high-level, we'll be building a key-value store (think <code>dict</code>) that has a <code>get</code> method that takes
an optional second parameter as a <code>time</code> object in Python to return the most recent value before
that period in time. If no key-value pair was added to the map before that period in time, return <code>None</code>.</p>
<p>For consistency’s sake, let’s call this class <code>TimedKVStore</code> and put it into a file called <code>kv_store.py</code>.</p>
<p>You’ll need some sort of <code>time</code> object to track when key-value pairs are getting added to this map.
Consider using the <code>time</code> module from Python <a href="https://docs.python.org/3.4/library/time.html" rel="nofollow">Docs</a></p>
<p>To give you an idea of how this class works, this is what should happen after you implement <code>TimedKVStore</code>.</p>
<pre><code>&gt;&gt;&gt; from kv_store import *

&gt;&gt;&gt; t0 = time.time()
&gt;&gt;&gt; d.put("1", 1)
&gt;&gt;&gt; t1 = time.time()
&gt;&gt;&gt; d.put("1", 1.1)
&gt;&gt;&gt; d.get("1")
1.1
&gt;&gt;&gt; d.get("1", t1)
1
&gt;&gt;&gt; d.get("1", t0)
None
